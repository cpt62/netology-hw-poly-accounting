# netology-hw-poly-company

Решение домашнего задания к занятию «Полиморфизм» в усложненном варианте.

1) Учитываю замечания Ф. Воронова, создаю отдельный репозиторий для размещения решения домашнего задания;
2) В качестве демонстрации работоспособности, в классе Main буду использовать следующую историю:

- Компания "Нетология" набирает 5 слушателей на курс;
- В качестве реализации, будет выступать продажа услуги обучения на курсе "Java разработчик с нуля"
- В качестве трат, будут фигурировать: ЗП преподавателя, реклама компании, хостинг сайта компании
- В классе `Company` в качестве демонстрации понимания такого "кита", как полиморфизм, выступают полиморфные методы:
  `payTaxes()`, `applyDeals(Deal[] deals)`.
- В методе `payTaxes()`, согласно условию необходимо после вывода уплаченного налога обнулить `debit`, `credit`.
  Для наглядности решил вывести данные значения перед очисткой.
